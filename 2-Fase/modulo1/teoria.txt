FUNDAMENTOS DE RED

1. Que es una red?

Una red informatica es un conjunto de dispositivos conectados entre si con e objetivo de intercambiar informacion
y compartir recursos mediante protocolos de comunicacion.

Los dispositivos pueden comunicarse utilizando medios fisicos, como cables, o medios inalambricos, como Wi-Fi.

Una red permite que diferentes equipos trabajen de forma conjunta y compartan informacion de manera eficiente. 





2. Para que sirve una red?

Las redes permiten:

* Compartir archivos.
* Compartir impresoras.
* Acceder a internet. 
* Compartir aplicaciones. 
* Enviar y recibir informacion.
* Permitir la comunicacion entre usuarios y dispositivos. 
* Administrar recursos de manera centralizada.





3. Componentes basicos de una red

Una red puede estar formada por diferentes dispositivos. 


HOST -> Es cualquier dispositivo conectado a una red que pueda enviar y recibir informacion. 

Ejemplos:

* Computador.
* Servidor.
* Celular.
* Camara IP.
* Impresora.


Cliente -> Es el dispositivo que solicita un servicio. 

Ejemplos:

* Abrir una pagina web.
* Descargar un archivo.
* Consultar una base de datos.


Servidor -> Es el equipo encargado de ofrecer servicios a otros dispositivos. 

Ejemplos: 

* Servidor Web.
* Servidor DNS.
* Servidor de archivos.
* servidor de correo.


Switch -> Dispositivo que conecta varios equipos dentro de una misma red local(LAN)

Su funcion es permitir la comunicacion eficiente entre los dispositivos conectados. 


Router -> Dispositivo encargado de conectar diferentes redes.

Normalmente conecta una red privada con internet y decide por donde debe enviarse la informacion.


Access Point (Punto De Acceso)

Permite que dispositivos inalambricos se conecten a una red mediante Wi-Fi. 


Firewall

Dispositivo o software encargado de controlar el trafico de red mediante reglas de seguridad.

Permite o bloquea conexiones dependiendo de la configuracion establecida.





4. Tipos De Redes

LAN (Local Area Network)

Red que conecta dispositivos dentro de un area pequena.

Ejemplos: 

* Casa
* Oficina 
* Colegio
* Universidad


WAN (Wide Area Network)

Conecta redes ubicadas en diferentes ciudades o paises.

El mejor ejemplo es el internet.


MAN (Metropolitan Area Network)

Conecta varias redes dentro de una misma ciudad.


WLAN (Wireless Local Area Network)

Es una LAN que utiliza comunicacion inalambrica mediante Wi-Fi.


VPN (VIrtual Private Network)

Permite crear una conexion segura y cifrada entre un usuario y otra red utilizando internet.

Es ampliamente utilizada por empresas para el acceso remoto. 





5. Comunicacion entre equipos

La comunicacion entre dispositivos ocurre mediante el intercambio de datos.

Ejemplo: 

Computador -> Switch -> Router -> Internet -> Servidor 

El servidor procesa la solicitud y responde siguiendo el camino inverso. 

Toda comunicacion necesita un origen, un destino y reglas que permitan el intercambio de informacion. 





6. Conceptos importantes 

Comunicacion

Proceso mediante el cual dos o mas dispositivos intercambian informacion.


Medios de transmision

Es el medio por donde viajan los datos. 

Pueden ser: 

* Cable Ethernet.
* Fibra Optica.
* Wi-Fi.


Protocolo

Conjunto de reglas que permiten la comunicacion entre dispositivos. 

Mas adelante se estudiaran protocolos como: 

* HTTP
* HTTPS
* DNS
* SSH
* FTP
* SMTP
