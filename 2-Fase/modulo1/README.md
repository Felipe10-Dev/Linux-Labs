# Módulo 1 - Fundamentos de Redes

## Descripción

En este módulo se estudian los conceptos fundamentales de las redes informáticas, los cuales constituyen una de las bases más importantes para cualquier profesional de ciberseguridad y, especialmente, para un Analista SOC.

El objetivo es comprender cómo se comunican los dispositivos, cuáles son los principales tipos de redes, qué función cumple cada dispositivo dentro de una infraestructura y cómo viaja la información entre equipos.

Al finalizar este módulo, el estudiante será capaz de identificar los componentes básicos de una red y entender el funcionamiento general de la comunicación entre dispositivos antes de profundizar en protocolos, direccionamiento IP y análisis de tráfico.

## Objetivos

* Comprender qué es una red informática.
* Identificar los principales tipos de redes.
* Conocer los dispositivos básicos de una red.
* Entender cómo se comunican los equipos.
* Construir una base sólida para los siguientes módulos de redes y ciberseguridad.

## Contenido

* ¿Qué es una red?
* Tipos de redes.
* Dispositivos de red.
* Comunicación entre equipos.

## Metodología

Cada tema del módulo estará compuesto por:

* Teoría.
* Laboratorio práctico.
* Misiones.
* Casos de uso orientados a SOC.
* Documentación en GitHub.

## Objetivo para un Analista SOC

Comprender los fundamentos de redes permite interpretar eventos de seguridad, analizar conexiones entre equipos, identificar el origen y destino del tráfico y entender cómo se comunican los sistemas dentro de una organización.
