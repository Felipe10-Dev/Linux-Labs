# Linux-Labs - Fundamentos de Linux para SOC Analyst

Colección de laboratorios prácticos enfocados en el aprendizaje de comandos esenciales de Linux utilizados en análisis de logs, investigación de incidentes y tareas de ciberseguridad.

## Objetivo

Desarrollar una base sólida en el uso de la terminal Linux mediante ejercicios prácticos orientados a entornos SOC (Security Operations Center), fortaleciendo habilidades de búsqueda, filtrado, procesamiento y análisis de información.

## Comandos estudiados

* find
* grep
* cut
* tr
* sed
* awk
* sort
* uniq
* wc

## Competencias desarrolladas

* Búsqueda de archivos y directorios.
* Filtrado de información dentro de logs.
* Extracción de campos y columnas.
* Transformación y limpieza de datos.
* Edición de texto desde terminal.
* Procesamiento de información por columnas.
* Ordenamiento de registros.
* Identificación de duplicados.
* Conteo y generación de estadísticas básicas.

## Enfoque

Cada módulo incluye:

* Teoría básica.

* Misiones de entrenamiento.
