-> GREP <-


¿Que es?
grep es un comando de Linux que permite buscar texto o patrones dentro de archivos.

¿Para que sirve?
- buscar palabras dentro de archivos
- filtrar lineas que contengan un patron
- contar coincidencias
- buscar ignorando mayusculas/minusculas
- trabajar con expresiones regulares (regex)


Sintaxis basica:
grep [opciones] "patron" archivo

Ejemplos: 
grep "Failed" auth.log -> busca la palabra Failed
grep "admin" users.txt -> busca la palabra admin
grep -i "failed" auth.log -> ignora mayusculas/minusculas
grep -w "root" users.txt -> busca palabra exacta
grep -n "Failed" auth.log -> muestra numero de linea


Opciones importantes: 



-> Fase 1 <-
-i = ignorar mayusculas/minusculas
-w = palabra exacta
-n = mostrar numero de linea
-c = contar coincidencias
-v = mostrar lo que no coincide


-> Fase 2 <-
-r = buscar recursivamente
-l = mostrar solo nombre de archivos
-h = ocultar nombre de archivo
-o = mostrar solo coincidencias
-m 1 = primera coincidencia por archivo


-> Fase 3 <- 
-E = usar expresiones regulares (regex)

Ejemplos regex: 
grep -E "[0-9]+" = buscar numeros
grep -E "[a-z]+" = buscar letras 
grep -E "[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+" = busca IPs

Uso en ciberseguridad:
Muy util para buscar errores, intentos fallidos, IPs sospechosas, usuarios comprometidos y patrones dentro de logs.
