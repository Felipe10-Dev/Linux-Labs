-> AWK <-

Que es?
awk es un comando de Linux que permite procesar y analizar texto por columnas o campos


Para que sirve? 
- extraer columnas
- filtrar informacion
- hacer comparaciones
- sumar valores
- generar reportes simples


Sintaxis Basica: 
awk 'accion' archivo


Ejemplos: 
awk '{print $1}' archivo.txt -> muestra columna 1
awk '{print $2}' archivo.txt -> muestra columna 2
awk -F ":" '{print $1}' users.txt -> usa ":" como delimitador
awk '{print $1,$3}' archivo.txt -> muestra columna 1 y 3 


Opciones Importantes:

-> Fase 1 <-

$1 = columna 1
$2 = columna 2
$3 = columna 3 


-F = define delimitador
print = imprimir columnas


-> Fase 2 <-

Comparaciones:
awk '$2 > 100' 

Condiciones:
awk '$2 == "Failed"'


-> Fase 3 <-

Operaciones: 

awk '{sum+=$3} END {print sum}'

Encadenamiento: 

grep "Failed" auth.log | awk '{print $7}'


Uso en ciberseguridad:
Muy util para analizar logs, extraer IPs, contar eventos, detectar patrones y generar reportes en entorns SOC.
