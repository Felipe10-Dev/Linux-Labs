-> CUT <-

¿Que es?
cut es un comando de Linux que permite extraer partes especificas de un texto, ya sea por columnas, campos o caracteres


¿Para que sirve?
- cortar columnas de un archivo 
- extraer campos usando delimitadores 
- seleccionar caracteres especificos
- filtrar informacion puntual dentro de logs

Sintaxis basica:
cut [opciones] archivo

Ejemplos:
cut -d ":" -f 1 users.txt -> extrae el primer campo usando ":" como delimitador
cut -d " " -f 4 auth.log -> extrae el cuarto campo
cut -c 1-5 texto.txt -> extrae caracteres del 1 al 5
cut -c 6- texto.txt -> extrae desde el caracter 6 hasta el final

Opciones Importantes: 

-> Fase 1 <-

-d = defie el delimitador
-f = selecciona campos
-c = selecciona caracteres


-> Fase 2 <-

-f 1,3 = selecciona varios campos
-c 1-5 = rango de caracteres
-c -5 = desde el inicio hasta el 5
-c 6- = desde el 6 hasta el final 


-> Fase 3 <-

Encadenamiento:
cut puede combinarse con otros comandos usando pipes "|"


Ejemplo:
grep "Failed" auth.log | cut -d " " -f 7


Uso en ciberseguridad:
Muy util para extraer IPs, usuarios, fechas, puertos y campos especificos dentro de logs.
